# Selenium Java Framework

A UI automation framework built with Selenium WebDriver, Java, TestNG, and Maven using the Page Object Model design pattern.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- WebDriverManager

## Features
- Page Object Model
- Reusable base classes
- Config-driven execution
- Explicit waits
- Clean project structure

## Test Coverage
- Valid login
- Invalid login

## Run Tests
```bash
mvn clean test
